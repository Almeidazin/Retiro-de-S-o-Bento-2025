As fotos estão organizadas por cartão de memória. Use o GitHub Pages para hospedar e compartilhar com todos.

Como compartilhar com todos

1 Hospede suas fotos no GitHub

Faça upload das suas fotos para um repositório no GitHub (pasta fotos/cartao01 e fotos/obra-prima)

2 Ative o GitHub Pages

No repositório, vá em Settings > Pages > Selecione a branch main e pasta /root > Save

3 Acesse o site

Seu site estará disponível em:

https://seuusuario.github.io/nome-do-repositorio

Compartilhe este link com todos!
