Como usar este site no GitHub Pages

  1. Faça upload deste arquivo HTML para seu repositório GitHub (pode nomear como index.html)
  2. Crie uma pasta chamada fotos no seu repositório
  3. Dentro da pasta fotos, crie as pastas cartao01 e obra-prima
  4. Faça upload das suas fotos para as pastas correspondentes
  5. Ative o GitHub Pages nas configurações do seu repositório
  6. Substitua os URLs das imagens no código JavaScript pelos caminhos reais das suas fotos
